


# Java OOP Concepts 

A structured collection of Java programs demonstrating the core **Object-Oriented Programming (OOP)** principles.  
This repository contains simple and practical examples to understand how OOP is implemented in real-world Java applications.

---

##  Concepts Covered

- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Method Overloading
- Method Overriding
